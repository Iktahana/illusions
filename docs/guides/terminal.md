# ターミナルの使い方

illusions では、エディタ内で直接シェル（PowerShell, bash, zsh など）を操作できる「統合ターミナル」を利用できます。

## ターミナルの開き方

1. **アクティビティバーから**: アクティビティバーの下部にあるターミナルアイコン（Terminal）をクリックします。
2. **コンテキストメニューから**: タブバーの空白部分や Dockview の空き領域を右クリックし、「新規ターミナル」を選択します。
3. **自動配置**: ターミナルを開くと、現在のアクティブなエディタの下部に分割して配置されます。

## 基本的な操作

- **フォーカス**: ターミナルパネル内をクリックすることで、入力をターミナルに向けることができます。
- **入力**: 通常のターミナルと同様にコマンドを入力して実行します。
- **リサイズ**: エディタとターミナルの境界線をドラッグすることで、サイズを調整できます。
- **閉じる**: ターミナルタブの「×」アイコンをクリックすると、シェルプロセスが終了し、タブが閉じられます。

## 設定

- **使用するシェル**: 設定画面の「ターミナル」タブから、OS にインストールされている利用可能なシェルから選択できます。
- **作業ディレクトリ (CWD)**: ターミナルは、プロジェクトモードではプロジェクトのルートディレクトリ、スタンドアロンモードではユーザーのホームディレクトリで起動します。

## 注意事項

- ターミナル機能は **デスクトップ版 (Electron)** 限定です。Web版ではご利用いただけません。
- 複数のターミナルを開くことができますが、リソース保護のため同時起動数に上限があります。
