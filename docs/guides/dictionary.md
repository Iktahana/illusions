# 校正の無視と辞書設定

illusions では、校正機能の指摘を管理するために、特定の箇所の指摘を無視する機能や、ダウンロード可能な辞書設定を利用できます。

## 校正の無視 (Ignored Corrections)

特定の箇所の指摘が著者の意図によるものである場合、その指摘を非表示にすることができます。

### 1. 個別に無視する

エディタ右側の「校正」パネル（インスペクター）に表示された各指摘カードの右上にある「目隠し（EyeOff）」アイコンをクリックします。

- これにより、その特定の箇所の指摘が無視リストに追加され、非表示になります。
- 文中で同じ単語が使われていても、他の箇所では引き続き指摘が表示されます。

### 2. 全ての同一指摘を無視する

「目隠し」アイコンを **右クリック** すると、そのファイル内で発生している同じ内容の指摘をすべて一括で無視リストに追加します。

## 辞書の設定

設定画面の「辞書」タブでは、校正エンジン（NLPバックエンド）が使用する辞書データの管理を行います。

- **辞書のダウンロード**: ウェルカム画面や設定から、最新の校正用辞書データをダウンロード・更新できます。

## ユーザー辞書（辞書外語の登録）

固有名詞・造語・専門用語などが「辞書外語（辞書に無い語）」として黄色の波線でマークされた場合、その語をユーザー辞書に登録して以後マークされないようにできます。

### 1. 語をユーザー辞書に追加する

辞書外語マークが付いた語に対し、次のいずれかの操作で **「この語をユーザー辞書に追加」** を実行できます。

- **校正パネル（インスペクター）**: 辞書外語の指摘カードに表示される「ユーザー辞書に追加」ボタンをクリックします。
- **エディタ上の波線**: 波線の付いた語を右クリックし、コンテキストメニューから「この語をユーザー辞書に追加」を選びます。

登録すると、その語の辞書外語マークは即座に消え、同じ語が文中の他の箇所に現れても以後マークされません。すでに登録済みの語を追加しようとした場合は、その旨が通知されます。

### 2. 登録のスコープ

ユーザー辞書はエディタのモードごとに保存先が分かれます。

| モード                   | 保存スコープ                               |
| :----------------------- | :----------------------------------------- |
| **プロジェクトモード**   | プロジェクト単位（`.illusions/` フォルダ） |
| **スタンドアロンモード** | ファイル単位（ローカルストレージ）         |

### 3. 既知語による抑制

ユーザー辞書に登録した語に加え、導入済みの辞書系ルールセットが提供する登録語も「既知語」として扱われ、辞書外語マークの対象から除外されます。これにより、自分の作品世界に固有の語彙を一度登録すれば、辞書外語の誤検出を継続的に抑えられます。

> 「ユーザー辞書に追加」操作は、辞書系の校正ルールセットがマニフェストで `suggestsDictionaryEntry` を宣言している場合に表示されます。

## 注意事項

- 無視した指摘を元に戻したい場合は、設定画面の「校正」セクションから無視リストをリセットできます。
- プロジェクト間で無視設定を共有するには、プロジェクトフォルダ内の `.illusions/` フォルダを含めて共有してください。
