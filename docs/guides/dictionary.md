# 校正の無視と辞書設定

illusions では、校正機能の指摘を管理するために、特定の箇所の指摘を無視する機能や、ダウンロード可能な辞書設定を利用できます。

## 校正の無視 (Ignored Corrections)

特定の箇所の指摘が著者の意図によるものである場合、その指摘を非表示にすることができます。

### 1. 個別に無視する

エディタ右側の「校正」パネル（インスペクター）に表示された各指摘カードの右上にある「目隠し（EyeOff）」アイコンをクリックします。

- これにより、その特定の箇所の指摘が無視リストに追加され、非表示になります。
- 文中で同じ単語が使われていても、他の箇所では引き続き指摘が表示されます。

### 2. 全ての同一指摘を無視する

「目隠し」アイコンを **右クリック** すると、そのファイル内で発生している同じ内容の指摘をすべて一括で無視リストに追加します。

## 辞書の設定

設定画面の「辞書」タブでは、校正エンジン（NLPバックエンド）が使用する辞書データの管理を行います。

- **辞書のダウンロード**: ウェルカム画面や設定から、最新の校正用辞書データをダウンロード・更新できます。
- **ユーザー辞書について**: 現在のバージョンでは、UI から個別の単語をユーザー辞書に直接追加する機能は制限されています。

## 注意事項

- 無視した指摘を元に戻したい場合は、設定画面の「校正」セクションから無視リストをリセットできます。
- プロジェクト間で無視設定を共有するには、プロジェクトフォルダ内の `.illusions/` フォルダを含めて共有してください。
