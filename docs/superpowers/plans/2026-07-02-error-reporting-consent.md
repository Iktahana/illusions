# Error Reporting Consent Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add privacy-first automatic error reporting for the Electron desktop app, with explicit user consent UI, GitHub Actions DSN injection, and no GitHub issue auto-filing yet.

**Architecture:** The app sends error reports only from trusted Electron integration points. Main process owns the GlitchTip DSN and final consent check. Renderer errors are sanitized and forwarded through a narrow IPC API or captured by renderer SDK only if the DSN is intentionally exposed as a public DSN. Privacy scrubbing is centralized and tested before any SDK event leaves the app.

**Tech Stack:** Electron, Next.js, React 19, TypeScript, Vitest, `@sentry/electron`, GlitchTip Sentry-compatible DSN, existing `StorageService`, existing `notificationManager`, GitHub Actions secrets.

### Out Of Scope Today

- Performance tracing.
- Log ingestion.
- User feedback widget.
- Adding secret management beyond the public GlitchTip DSN.

```text

```

- Do not commit the DSN value to the repository.
- Do not add GitHub issue tokens, GlitchTip admin API keys, or source-map upload tokens in this iteration.

## Privacy Contract

Never send:

- Manuscript content.
- File names.
- File paths.
- Project names.
- Search text.
- Dictionary contents.
- AI prompts or responses.
- User account email/name.
- DOM text.
- Breadcrumb data derived from UI clicks, console logs, network request bodies, or paths.

Allowed:

- Error type.
- Sanitized error message.
- Sanitized stack frame function/module names when paths are stripped.
- App version.
- Platform.
- Process type (`main` / `renderer`).
- Error boundary section name, if generic Japanese UI labels only.

Default SDK options:

```ts
sendDefaultPii: false;
tracesSampleRate: 0;
maxBreadcrumbs: 0;
```

```tsx
"use client";

import { useEffect } from "react";

export function ErrorReportingRuntime(): null {
  useEffect(() => {
    const onError = (event: ErrorEvent): void => {
      void window.electronAPI?.errorReporting?.captureRendererError({
        source: "window-error",
        name: event.error instanceof Error ? event.error.name : undefined,
        message: event.message,
        stack: event.error instanceof Error ? event.error.stack : undefined,
      });
    };

    const onUnhandledRejection = (event: PromiseRejectionEvent): void => {
      const reason = event.reason;
      void window.electronAPI?.errorReporting?.captureRendererError({
        source: "unhandledrejection",
        name: reason instanceof Error ? reason.name : undefined,
        message: reason instanceof Error ? reason.message : String(reason),
        stack: reason instanceof Error ? reason.stack : undefined,
      });
    };

    window.addEventListener("error", onError);
    window.addEventListener("unhandledrejection", onUnhandledRejection);
    return () => {
      window.removeEventListener("error", onError);
      window.removeEventListener("unhandledrejection", onUnhandledRejection);
    };
  }, []);

  return null;
}
```
