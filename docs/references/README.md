# リファレンス

ここには、実装仕様そのものではないが、設計判断や表記ルールの根拠になる資料を集約します。

## 想定している内容

- 日本語組版・縦書き・禁則処理に関する外部規格
- MDI 設計時に参照した表記・排版ガイド
- UI/UX や編集体験に関する補助メモ

## 運用方針

- 正式な仕様は `architecture/`、`guides/`、`MDI/` に置く
- 外部資料への索引や要約だけをここに置く
- 単発の検証ログは `archive/` に逃がす
