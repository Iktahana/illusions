# Branding source assets

This directory contains editable source files used to generate public and store branding assets.

- Files under this directory are not served by Next.js and are not included in the web runtime.
- Runtime PNG/SVG/ICO assets remain under `public/`, `build/`, and `store/`.
- After editing a source file, regenerate the derived assets with the relevant script in `scripts/` and commit both the source and intentional outputs.
