illusions は、日本語小説を書く人のための執筆エディタです。

縦書きやルビに対応した書きやすい編集環境に、文章校正、読み上げ、執筆分析をまとめました。
下書きから推敲まで、アプリを切り替えずに進められます。

主な特長

・縦書き、ルビ、縦中横に対応し、日本語小説らしい体裁で書けます
・22項目の文章校正で、表記ゆれや文体の乱れに気づけます
・読み上げ機能で、文章のリズムや不自然な言い回しを耳で確認できます
・自動保存と履歴管理で、書き直しや復元がしやすくなります
・原稿はパソコン内に保存され、クラウドへ自動送信されません

執筆機能
・縦書き・横書きを切り替えられます
・ルビ（ふりがな）を振れます
・縦中横で数字や英数字を縦書き内に横向き配置できます
・2秒ごとの自動保存で、手動保存の手間がありません
・予期しない終了時も、次回起動時に直前の状態を復元します

文章校正
・文化庁の公用文基準に基づく22項目をリアルタイムで確認できます
・句読点の使い方、常用漢字、数字表記などをチェックします
・文体の混在（です・ます調 / だ・である調）を検出します
・接続詞や受け身表現の使いすぎを指摘します

読み上げ
・書いた文章を音声で再生し、リズムや語感を確認できます
・速度・音程・音量を調整できます
・読み上げ中の箇所がハイライトされます

執筆分析
・文字数、段落数、原稿用紙換算をリアルタイムで表示します
・品詞別の色分け表示で、文の構造を確認できます
・単語の出現頻度を分析し、繰り返しを減らせます
・読みやすさの評価と読了時間の目安を表示します

エクスポート
・PDF、EPUB、DOCX、TXT に書き出せます

プライバシー
・原稿データはすべてお使いのパソコン内に保存されます
・クラウドへの自動送信はありません
・ソースコードは GitHub で公開しています
