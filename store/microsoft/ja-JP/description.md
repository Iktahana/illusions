【illusions - 日本語小説執筆専用エディタ】

illusions は、日本語小説の執筆を強力にサポートする専門的なエディタです。
縦書き、ルビ（ふりがな）、AI校正など、小説家に必要なすべての機能を搭載しています。

✨ 主な機能

📝 執筆機能
・縦書き表示対応 - 日本の小説らしい体裁で執筆
・ルビ（ふりがな）のサポート - 難しい漢字にルビを振れます
・縦中横 (TCY) - 数字や英数字を縦書きで横にできます
・究極の自動保存 (Auto-save) - 執筆中、2秒ごとにバックアップを自動生成。 手動保存の煩わしさから解放され、創作だけに没頭できます。
・堅牢なクラッシュリカバリ - 万が一のPCフリーズや予期せぬ終了時も、次回の起動時に直前の状態を完璧に復元。
・不意のミスを防ぐ履歴管理 - 2秒ごとの保存に加え、バージョン履歴機能で「書き直す前の方が良かった」という後悔もゼロに。

🤖 AI校正・校閲
・ローカルLLM搭載のAI校正 - クラウド送信なし、プライバシーを守ります
・22個の日本語文法ルール - 句点、用字用語、JIS基準に準拠
・文品質検査 - 常用漢字、改行位置、約物のチェック

📊 執筆分析
・文字数・段落数・原稿用紙換算 - 執筆進捗の可視化
・品詞別ハイライト - 日本語の文法構造を色分けで表示
・単語頻度分析 - 繰り返し表現をチェック
・差分表示機能 - 過去の版との変更部分を確認

💾 保存・同期・共有
・複数フォーマット対応 - PDF、EPUB、DOCX にエクスポート
・バージョン履歴 - すべての変更を保存、いつでも過去の版に戻せます

🎨 快適な執筆環境
・ダークモード / ライトモード自動切り替え
・複数のフォント・文字サイズ対応
・行間・字間の細かい設定

🖥️ クロスプラットフォーム対応
・Windows、macOS で動作
・Chrome ブラウザ版も無料で利用可能（https://illusions.app/）

📱 セキュリティ & プライバシー
✅ すべてのデータはお使いのパソコンに保存
✅ AI処理もローカルで実行 - インターネット送信なし
✅ オープンソース - ソースコードが GitHub で公開されており、安全性を確認できます

🎯 こんな方にお勧めします

・日本語の文法やスタイルを改善したい方
