# Background Images

Place background images in this directory with the naming convention:
- `bg1.jpg`
- `bg2.jpg`
- `bg3.jpg`
- `bg4.jpg`
- `bg5.jpg`

## Configuration

Update the `BG_COUNT` constant in `src/main.ts` to match the number of background images.

```typescript
const BG_COUNT = 5; // Update this number
```

## Recommended Specifications

- Format: JPEG or PNG
- Resolution: 1920x1080 or higher
- File size: < 500KB (optimized for web)
- Aspect ratio: 16:9 or wider
